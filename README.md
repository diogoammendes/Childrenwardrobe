# Children Wardrobe

Aplicação web para gestão de guarda-roupa infantil. Permite aos pais gerir o guarda-roupa dos seus filhos, adicionar peças de roupa, acompanhar tamanhos e muito mais.

## Funcionalidades

- **Área de Administração**: Gestão de utilizadores e sistema
- **Gestão de Crianças**: Adicionar e gerir informações das crianças
- **Guarda-Roupa**: Adicionar, editar e organizar peças de roupa por categoria
- **Transferência de Itens**: Transferir peças entre crianças da mesma família
- **Controlo de Stock**: Definir mínimos por categoria e subcategoria

## Tecnologias

- Next.js 14
- TypeScript
- Prisma (PostgreSQL)
- Tailwind CSS
- NextAuth.js

## Desenvolvimento

```bash
npm install
npm run dev
```
